QRS波群检测是动态心电图系统心律失常分析的核心基础，YY 9706.247-2021标准对其检测准确性、测试方法、性能指标及试验条件均有明确且严格的规定，核心要求是通过标准化数据库验证，确保敏感度和阳性预测值满足临床诊断需求。

### 一、检测核心定义与测试依据
1. **QRS波群定义**：心室除极阶段心电图呈现的波形（201.3.216），是区分正常心搏与异位搏动的关键标识。
2. **强制测试数据库**：
   - 必选数据库：AHA（80份记录，含室性心律失常）、MIT-BIH（48份记录，含常见/罕见心律失常）、NST（12份记录，含噪声抑制测试）。
   - 排除规则：剔除6份含起搏心搏的记录（AHA 2份、MIT-BIH 4份），NST中3份纯噪声记录不参与标准试验。
   - 测试周期：每份记录前5min为算法“学习期”（不计入统计），剩余时间为测试期，需连续输入数据（禁止倒带/快进）。

### 二、检测准确性的测试方法：搏-搏比对
标准规定通过“搏-搏比对”验证QRS波群检测的准确性，核心是将算法检测结果与数据库参考注释（心脏病专家标注）逐一心搏匹配（201.12.1.101.2.3）。

#### 1. 比对核心规则
- 匹配窗口：算法检测的心搏时间与参考心搏时间差的绝对值≤150ms，视为有效匹配。
- 标记分类：参考注释与算法结果均需按心搏类型标记（N=正常心搏、S=室上性异位、V=室性异位、F=融合心搏、Q=无法分类心搏等），匹配结果计入矩阵（表201.105）。
- 伪心搏处理：未匹配的参考心搏视为“漏检”（标记X/O），未匹配的算法检测结果视为“假阳性”（标记O/X），均参与统计。

#### 2. 比对流程
1. 初始化：设学习期后首个参考心搏时间为T、首个算法检测心搏时间为t，比对矩阵清零。
2. 逐一心搏匹配：
   - 若t在T的150ms内且更接近T，视为匹配，更新T和t至下一个心搏。
   - 若t先于T且超出匹配窗口，算法结果记为“假阳性”，更新t至下一个。
   - 若T先于t且超出匹配窗口，参考心搏记为“漏检”，更新T至下一个。
3. 矩阵统计：匹配结果按“参考标记-算法标记”组合计入矩阵（如Nn=正常心搏正确检测、Nv=正常心搏误判为室性异位）。

### 三、核心性能指标与计算方法
QRS波群检测的核心性能指标为“敏感度（Se）”和“阳性预测值（+P）”，需按总量统计（每个心搏同等权重）和平均统计（每个记录同等权重）分别报告（201.12.1.101.1.5.2）。

#### 1. 指标定义与公式
- **QRS敏感度（QRS Se）**：正确检测的QRS波数占总参考QRS波数的比例，反映“不漏检”能力。
- **QRS阳性预测值（QRS +P）**：正确检测的QRS波数占算法检测总QRS波数的比例，反映“不错检”能力。
- 计算公式（基于比对矩阵）：
  \[
  QTP = Nn + Ns + Nv + Nf + Nq + Sn + Ss + Sv + Sf + Sq + Vn + Vs + Vv + Vf + Vq + Fn + Fs + Fv + Ff + Fq + Qn + Qs + Qv + Qf + Qq
  \]
  \[
  QFN = No + Nx + So + Sx + Vo + Vx + Fo + Fx + Qo + Qx
  \]
  \[
  QFP = On + Os + Ov + Of + Oq + Xn + Xs + Xv + Xf + Xq
  \]
  \[
  QRS\ Se = \frac{QTP}{QTP + QFN}, \quad QRS\ +P = \frac{QTP}{QTP + QFP}
  \]

#### 2. 指标要求
- 数据库强制要求：AHA、MIT-BIH数据库需报告“必填项（R）”，NST数据库对QRS敏感度为必填、阳性预测值为可选。
- 无明确数值阈值，但需通过标准数据库测试，且统计结果需与临床可接受水平一致（附录AA示例中总统计敏感度/阳性预测值均≥95%）。

### 四、影响检测准确性的关键技术要求
1. **信号处理性能**：
   - 输入阻抗≥10MΩ（10Hz测试频率，覆盖±300mV直流偏置），减少电极接触噪声影响。
   - 共模抑制比≥60dB（网电源频率）、≥45dB（2倍网电源频率），抵抗工频干扰。
   - 系统底噪≤50μV（峰-谷值），避免噪声误判为QRS波。
2. **时间与增益稳定性**：
   - 时间准确度：24h内总误差≤30s，确保心搏时间标记精准。
   - 增益稳定性：24h内增益变化≤3%，避免QRS波幅度失真导致漏检/误检。
3. **多通道协同**：
   - 通道间时间轴对齐误差≤±20ms（25mm/s走纸速度），多导联数据融合时不影响QRS波定位。

### 五、试验环境与预处理要求
1. **预处理公开**：若对数据库信号进行重采样、格式转换、增益调整等预处理，需详细公开步骤，确保第三方可复现。
2. **自动增益控制（AGC）**：模拟信号测试时，AGC稳定后再开始检测；数字信号测试时，需模拟AGC功能，避免增益波动影响QRS波识别。
3. **无人工干预**：测试全程禁用用户手动调整分析模式的功能，确保算法自主检测性能。
